GoSlow

A video booth interface for connecting to and controlling a GoPro camera over WiFi.

History

GoSlow is a video booth originally designed to capture short, high speed footage. After seeing the Slow Motion Booth, we wanted to make the same type of video for a friend's wedding, but we also wanted to enjoy the wedding.

The video booth had to use hardware we already had, so it was built on an iPad 2 and a GoPro Hero 3 Black. It also had to be fully automated so that we only had to visit it every so often to make sure it didn't crash. In the end we came up with this video, and we couldn't be happier.
